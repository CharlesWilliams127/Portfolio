Enhanced Portfolio Summary (For making grading easier)

My main issues from project 2 were CSS validation errors, those have all been fixed.

My SPA, or DHTML, is located on my main portfolio page. Clicking on either thumbnail on the page will dynamically build the content that the thumbnail cooresponds to.

I am using several Javascript animations, the arrow buttons that close each section or point to the About Me section are all two seperate images that adjust opacity through Javascript. Each piece of content that dynamically builds will also fade in, and then when it is removed it will fade out.

There is a gallery on every project page (except Asteroids since I only have one image), each gallery uses the FancyBox plugin and is activated by clicking on any of the images on the page.

All contact info is located in the About Me section

Both the Authorization page and the 404 error page can be accessed through links at the top of this page.
